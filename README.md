## 🎯 What this does

Extract text from images, recognize handwritten digits, translate to any language, and listen to the results.

### Features
- 🖼️ **OCR**: Tesseract or EasyOCR (choose your preference)
- 🧠 **Neural Network**: GRNN model with 97% accuracy
- 🌍 **Translation**: English, French, Arabic support
- 🔊 **Text-to-Speech**: Hear results in multiple languages
- 🎨 **GUI**: Easy-to-use Tkinter interface

## 🚀 Quick Start

```bash
pip install -r requirements.txt
python test_interface.py
```

### How to Use
1. Click "Importer une image" (Import image)
2. Click "Afficher le texte" (Show text)
3. Click translation button (Arabic, French, or English)
4. Click "Lire le texte" (Read text aloud)
